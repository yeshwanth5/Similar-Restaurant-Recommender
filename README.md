# Similar-restaurant-recommender
Unsupervised learning model that recommends similar restaurant types in Bangalore
Dataset - https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants?datasetId=153420&searchQuery=Clust

NearestNeighbour machine learning model applied on cleaned zomato bangalore dataset. 
CountVectorizer and Geopy also used to vectorize cuisines and to geocode the locations in Bangalore. 
A small generic web UI is made with the server running on python Flask.
